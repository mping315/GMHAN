# GMHAN
## Abstract

## Data
The data used in this study is saved in the './data/' folder. Among them, the gene data is stored in the './data/gene/' folder, and the miRNA data is stored in the './data/miRNA/' folder.


## Requirements
GMHAN codes is baesd on Pytorch and Python and DGL library. So you will need the following packages to run.  
+ Python==3.9.21
+ torch==2.3.1
+ dgl==2.4.0
+ torch-geometric==2.6.1
+ torchvision==0.18.1

## Overview
Here, we provide an implementation of GMHAN built with Pytorch and Pytorch Genometric. The repository structure is organised as follows:
-model.py: Defines the core architecture of GMHAN. 
-utils.py: Functions used in GMHAN
-train_and_test.ipynb: Train and test the model.
